
## Editors, themes and fonts I use

### Sublime
I use Sublime a lot for writing documentation and text editing. I'm currently using the theme [Monokai Pro](https://monokai.pro/) but I've also used [Ayu](https://packagecontrol.io/packages/ayu) and [Dracula](https://draculatheme.com/).

### Visual Studio Code
I use VsCode mostly for coding with JavaScript, Go, Rust and Python. I'm currently using the [Dracula Pro](https://draculatheme.com/pro/) theme and I love it!

### PHPStorm
In my day to day at work I code mostly in PHP and PHPStorm is my editor of choice for it. I'm currently using awesome [CodelyTV theme](https://plugins.jetbrains.com/plugin/12891-codelytv-theme).

### Fonts
Currently I'm using the Operator Mono for all my editors and [Consolas](https://www.fontpalace.com/font-details/Consolas/) for my terminal and other UIs but I've also used [JetBrains Mono](https://www.jetbrains.com/lp/mono/) and [Fira Code](https://github.com/tonsky/FiraCode).
